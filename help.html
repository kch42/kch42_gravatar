gravatar

This plugin gives you an STE tag, which will return the Gravatar of the person who wrote a comment.

This tag is: <ste:gravatar />.

How to use this tag

The tag will return a HTML <img /> tag.

The tag will accept these parameters:

Parameter name Mandatory? Default value Description
comment Yes   Name of a variable that contains information about a comment (usually generated by <ste:article_comments />)
size No 80 Size of image in pixels.
default No identicon The default image, if no Gravatar is available. Can be a URL to an image or one of Gravatar's default values.
rating No g One of Gravatar's rating values.
style No   Additional (CSS) styles for the image.
